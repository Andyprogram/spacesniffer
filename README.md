## 🔧 核心算法

### Squarified Treemap（Bruls, Huizing, van Wijk, 2000）

```
输入: 文件/文件夹列表 (含字节大小)，包围矩形 (x, y, w, h)
输出: 每个项目的位置和尺寸

1. 归一化: 所有项目大小按比例缩放到包围矩形面积
2. 排序: 按大小降序排列
3. 递归布局:
   a. 沿包围矩形短边方向排列
   b. 贪心构建"行": 逐个添加项目，直到宽高比不再改善
   c. 固定当前行布局，在剩余空间递归处理剩余项目
4. 宽高比公式:
   horizontal: worst = max(norm_i × w² / S²,  S² / (norm_i × w²))
   vertical:   worst = max(norm_i × h² / S²,  S² / (norm_i × h²))
   (worst 越低越好，1.0 = 完美正方形)
```

相比传统的 **slice-and-dice**（单向切片），squarified 算法产生的矩形长宽比分布更均匀，绝大多数矩形接近正方形，极大地提升了可读性和交互体验。

## 🏗️ 技术栈

| 层级 | 技术 |
|------|------|
| 语言 | Python 3.14 |
| GUI 框架 | tkinter（标准库） |
| 并发 | `threading` + `queue`（标准库） |
| 文件系统 | `os.scandir()` 高性能遍历 |
| 打包工具 | PyInstaller 6.19（`--onefile --windowed`） |
| 平台 | Windows 11（兼容 Windows 10+ / Linux / macOS） |

> **为什么选 Python + tkinter？**
> - tkinter 随 Python 发行，无需安装任何第三方 GUI 库
> - `os.scandir()` 是系统级目录遍历，性能优秀
> - PyInstaller 打包后为单个 12MB 的 exe，分发简单
> - 跨平台兼容（Windows / Linux / macOS 均可运行）
